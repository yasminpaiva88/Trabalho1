Sobre o Projeto

O IncluiWeb é uma ferramenta fictícia voltada para promover a inclusão digital, auxiliando desenvolvedores e empresas a avaliar e melhorar a acessibilidade de seus sites e aplicações.

Com uma interface intuitiva e relatórios detalhados, o sistema fornece recomendações práticas baseadas nas Diretrizes de Acessibilidade para Conteúdo Web (WCAG).

Principais Recursos

  • Análise automática de contraste de cores
  • Verificação de uso de elementos semânticos
  • Relatórios com sugestões de correção
  • Tutoriais sobre acessibilidade na web
